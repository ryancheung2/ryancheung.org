Chrome Extensions

GmailPro

GmailPro is a Chrome extension that allows you easily customize Gmail.

Some of the functions included the the free version of GmailPro are: Light Mode, Compact Mode, and Border Mode.

GmailPro Premium

GmailPro Premium is a paid version of GmailPro that includes all the functions of the free version. GmailPro Premium also includes more features and customizations.

Some of the functions included in the premium version of GmailPro are: Custom Colors, Email Templates, and more Shortcuts.

The premium version of GmailPro is available for purchase on the Chrome Web Store.
